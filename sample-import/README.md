# Sample Question Import Kit

Files:

- `sample_questions.xlsx` — recommended formatted template.
- `sample_questions.csv` — the same eight questions in CSV form.
- `images/` — SVG diagrams referenced by `question_image`.

Important rules for future uploads:

1. Keep every `question_id` unique.
2. Use `A`, `B`, `C`, or `D` in `correct_answer` for a single-correct MCQ.
3. Write LaTeX without surrounding dollar signs.
4. Put only the filename in `question_image`, not a computer path.
5. Keep image filenames unique.
6. Do not merge cells in the import data area.
7. Do not rename the mandatory column headers.
8. Version 3 will add the browser upload and validation engine. Version 1 includes this kit so the data format can be tested and refined now.
