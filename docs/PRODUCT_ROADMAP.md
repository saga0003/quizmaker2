# Product Roadmap

## V1 — Functional pilot

A complete walkthrough for super admin, organisation admin, school admin and student users. This release validates the information architecture, school-sales demonstration flow and student analytics presentation.

## V2 — Connected core

1. Supabase authentication with role-based access.
2. Tenant isolation for organisations, schools and branches.
3. Student, parent and staff onboarding.
4. Question bank with board, grade, subject, chapter, topic, cognitive skill, difficulty and expected time.
5. Assessment blueprint builder and randomised delivery.
6. Autosaved responses, timers and attempt events.
7. Real analytics from completed attempts.

## V3 — School delivery

1. OMR paper and answer-sheet generation.
2. Scan/import pipeline.
3. Parent OTP report access.
4. Teacher intervention groups.
5. Bilingual reports.
6. Odoo CRM integration for explicit counselling opt-ins.

## V4 — Differentiation

1. Item-response and psychometric monitoring.
2. Adaptive daily practice.
3. Forgetting-curve retesting.
4. District and state benchmark challenges.
5. Scholarship and verified recognition engine.
6. AI-assisted report explanations with teacher review.

## Non-negotiable principles

- Academic ability and enrolment probability are separate scores.
- Performance data never grants marketing permission.
- No deterministic prediction that a minor will or will not clear an examination.
- Every report must provide a specific next learning action.
- Teacher judgement overrides automated intervention suggestions.
