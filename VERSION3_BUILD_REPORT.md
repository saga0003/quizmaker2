# ScholarOS — Version 3 Build Report

## Added

- Unique username registration with immediate access when Supabase email confirmation is disabled.
- Google account selector retained.
- ScholarOS master and school-private question banks.
- Manual question editor.
- Single-correct, multiple-correct, numerical, integer, assertion-reason, match, passage and image question models.
- LaTeX previews for questions, options and solutions.
- Direct image upload to Supabase Storage and public image URL support.
- CSV and XLSX browser import without paid services.
- ZIP image matching and upload.
- Row validation, duplicate-in-file warning and import summaries.
- Subject and chapter taxonomy.
- Review queue with approve, reject and request-changes actions.
- Question version snapshots and audit entries.
- Excel/CSV templates and sample image ZIP.
- Admin and school question-bank routes.
- Hostinger static-export script with route and CSS validation.

## Next version

Version 4 will use approved questions to build configurable NEET, JEE, KCET and custom examinations, including paper blueprints, scheduling, timers, autosave, response storage and evaluation.
